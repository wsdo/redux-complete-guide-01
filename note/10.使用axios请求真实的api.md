## 10.使用axios请求真实的api

```
export const getArticle = () =>{
  return dispatch =>{
    setTimeout(()=>{
      // dispatch(eat('煎饼')) 
      dispatch(()=>{
        axios.get('/v1/article/list').then(res=>{
          console.log('article',res)
        })
      }) 
    },2000)
  }
}
```
